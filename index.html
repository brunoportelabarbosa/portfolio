Sobre mim

Olá! Meu nome é Bruno Portela Barbosa e sou um entusiasta da tecnologia.

Atualmente sou supervisor técnico em um Centro de Diagnóstico por Imagem, liderando equipe e gerenciando projetos de melhorias na unidade de saúde pública.

Portfólio

Projeto 1: Sistema de Exames

Elaborado e desenvolvido um sistema de exames em php para gerenciar pedidos de correções, adiantamento de laudos, reconvocações, etc. O sistema é utilizado por toda unidade abrangendo todos os setores, onde as solicitações são encaminhas para o setor de tecnologia onde fazem os ajustes necessarios e liberações solicitadas.


Projeto 2: Sistema de Estacionameno

Sistema montado em php para gerenciar toda movimentação do estacionamento da unidade de saúde onde trabalho. O sistema apresenta uma lista dos veiculos estacionados no momento, um painel informando o tempo de permanência. Dashboard com veiculos com mais entradas, dias com maior movimentação, entre outros.


Projeto 3: Intranet

Mais um sistema para melhoria da unidade, também montado em php. A intranet possui uma area inicial com cards de postagens com informações relevantes, um card com aniversariantes do mês, eventos da unidade. Possui area com lista de ramais, pasta de documentos públicos para download, area para exibir escala dos colaboradores de cada setor, entre outras melhorias.
